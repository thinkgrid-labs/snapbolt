Client-Side Image Optimization

Powered by Rust compiled to WebAssembly. No framework, no build step, no server — just a .wasm file and plain JavaScript.

Pick a JPEG or PNG. Snapbolt resizes and converts it to WebP entirely in your browser using the Rust WASM encoder.
